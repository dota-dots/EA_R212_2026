Points clés

1. [localStorage]

La propriété localStorage permet de stocker ou de recuprer des chaines de caractères selon la fonction (localStorage.setItem ou localStorage.getItem).

2. [Filtrage interactif]

Le filtrage interactif est essentiel dans un site web afin de trier de elements. Pour ce faire nous avons les boutons "class="filtre"  avec le tag approprié, qu'ainsi des fonctions pour mettre à jour le bouton actif et filtrer les données.

3. [méthode forEach()]

La méthode forEach() permet d'exécuter une fonction donnée sur chaque élément d'un tableau. Elle permet de facilité le cheminement sur un tableau sans avoir besoin d'utiliser d'une boucle.
